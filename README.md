# Pinlatro

This is Pinlatro! A dynamic and unique blend of classic pinball and roguelike mechanics, with a heavy emphasis on RNG-based build making.

Please note that this game is a proof of concept and is currently not being actively developed. While the core gameplay loops and concepts are fully implemented, the project remains a raw prototype without much visual or technical polish.
